# 📊 Investment Data Analysis & Visualization (Power BI Style)

## 🔍 Overview
This project was completed as part of my Power BI internship at Cognifyz Technologies. The goal was to analyze investment-related data and understand how people make financial decisions.

Instead of using Power BI directly, I implemented the complete workflow using Python in a Jupyter Notebook, following a dashboard-style approach. The project is divided into 7 structured tasks, each focusing on a different aspect of data analysis and visualization.

---

## 🏢 Internship Context
During this internship, I worked on real-world style tasks related to:
- Data analysis  
- Data visualization  
- Understanding user behavior through data  
- Presenting insights in a structured way  

This project reflects my hands-on learning and practical implementation of these concepts.

---

## 🎯 What I aimed to do
- Explore and understand the dataset  
- Clean and prepare the data  
- Identify patterns in investment behavior  
- Visualize insights using graphs  
- Combine everything into a dashboard-style analysis  

---

## 📌 What I did (Task-wise)

### 🔹 Task 1: Data Exploration and Summary  
I started by importing the dataset and understanding its structure. I calculated basic statistics like average age and explored investment preferences. Then I created visualizations to summarize the data.

---

### 🔹 Task 2: Gender-based Analysis  
Here, I compared how investment preferences differ between genders. I used charts to clearly show trends in equity, mutual funds, and other options.

---

### 🔹 Task 3: Objective Analysis  
In this step, I analyzed how people's financial goals (like savings or growth) influence their investment choices. This helped in understanding decision patterns.

---

### 🔹 Task 4: Investment Duration & Frequency  
I looked at how long people invest and how frequently they monitor their investments. This gave insights into investment behavior over time.

---

### 🔹 Task 5: Reasons for Investment  
I explored why people invest — whether it's for better returns, tax benefits, or other reasons — and visualized the most common motivations.

---

### 🔹 Task 6: Source of Information  
This task focused on where people get their investment knowledge from, such as the internet or newspapers. It helped identify the most trusted sources.

---

### 🔹 Task 7: Dashboard Integration  
Finally, I combined all the insights into a structured, dashboard-style analysis. The goal was to make the results easy to understand, similar to a Power BI report.
